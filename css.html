1. Listy

Znaki/punktacja listy
list-style-type: wartość;

Przykłady
none - brak numeracji/znaku
disc - wypełnione kółka
circle - obręcze
square - kwadraty
inherit - przyjmuje styl swojego "rodzica"


Grafika jako punktacja listy
list-style-image: wartość;

Przykłady
none - brak obrazka
url - obrazek z podaj ścieżki
initial - ustawia styl domyślny
inherit - przyjmuje styl swojego "rodzica"
