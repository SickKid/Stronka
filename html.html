1. Sekcja header

title
- pozwala na zmianę tytułu strony, który jest wyświetlany na górnym pasku przeglądarki
<title>TYTUŁ STRONY</title>


meta
- znacznik odpowiedzialny za metadane strony
<meta charset="UTF-8">
<meta name="description" content="Opis strony">
<meta name="keywords" content="Słowa, kluczowe, tagi">
<meta name="author" content="Autor">
<meta name="viewport" content="width=device-width, initial-scale=1.0">


link
- służy do podpięcia stylu, który znajduje się poza plikiem html
<link rel="stylesheet" type="text/css" href="style.css" />


script
- służy do podpięcia skryptu, który znajduje się poza plikiem html
<script src="ścieżka"></script>

Skryptu można użyć również bezpośrednio w pliku html
<script>SKRYPT</script>




2. Sekcja body

Atrybuty Główne

class
- Określa jedną lub więcej nazw klas dla elementu (dotyczy klasy w arkuszu stylów)
<div class="nazwa klasy"></div>


id
- określa unikatową nazwę danego elementu
<div id="nazwa"></div>


style
- określa wbudowany styl CSS dla elementu
<span style="atrybut CSS: wartość atrybutu"><span>


title
- określa dodatkowe informacje o elemencie
<p title="informacje"></p>




Atrybuty zdarzeń okna

onerror
- skrypt przeznaczony do uruchomienia w wypadku błędu
<img src="ścieżka/do/pliku" onerror="funkcja()">


onload
- skrypt przeznaczony do uruchomienia po załadowaniu strony
<body onload="funkcja()">
